Getting Started
===============

Folder Structure
----------------

VRUB expects addon components to follow a specific folder structure, which you can see below::

    addon_name
    |   manifest.json
    |   preview_image.jpg
    |   workshop.json
    |   
    \---overlays
        +---overlay_name
        |       index.html
        |       manifest.json
    \---plugins
        +---plugin_name
        |       PluginName.dll
        |       manifest.json
